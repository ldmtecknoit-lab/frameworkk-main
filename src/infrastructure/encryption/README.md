# Encryption Adapters (`/src/infrastructure/encryption`)

Questa directory contiene gli adattatori per la crittografia e la sicurezza dei dati.

## File Principali

*   **`aes.py`**: Implementazione dell'algoritmo AES (Advanced Encryption Standard) per la crittografia simmetrica.
*   **`rsa.py`**: Implementazione dell'algoritmo RSA per la crittografia asimmetrica.
*   **`hashing.py`**: Funzioni di hashing generiche.
*   **`sha512.py`**: Implementazione specifica dell'hashing SHA-512.
