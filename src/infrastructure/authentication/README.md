# Authentication Adapters (`/src/infrastructure/authentication`)

Questa directory contiene gli adattatori per l'autenticazione degli utenti.

## File Principali

*   **`supabase.py`**: Adattatore per Supabase Auth. Gestisce login, registrazione e gestione sessioni tramite Supabase.
*   **`oauth.py`**: Adattatore generico per protocolli OAuth2 (es. Login con GitHub, Google).
