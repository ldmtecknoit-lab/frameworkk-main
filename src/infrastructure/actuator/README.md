# Actuator Adapters (`/src/infrastructure/actuator`)

Questa directory contiene gli adattatori per gli "Attuatori", ovvero componenti che eseguono azioni nel mondo reale o su sistemi esterni.

## File Principali

*   **`ansible.py`**: Integrazione con Ansible per l'automazione DevOps e il deployment.
*   **`flow.py`**: Probabilmente gestisce flussi di lavoro o pipeline di esecuzione.
*   **`ssh.py`**: Esecuzione di comandi remoti via SSH.
*   **`console.py`**: Output su console.
*   **`motor.py`, `speaker.py`**: Adattatori per hardware fisico (motori, altoparlanti), utili in contesti IoT o robotica.
*   **`mouse.py`**: Automazione dell'input mouse.
