# Framework Schemes (`/src/framework/scheme`)

Questa directory contiene le definizioni degli schemi dati e delle configurazioni, principalmente in formato JSON.

## Contenuto

Questi file definiscono la struttura dei dati utilizzati dai vari componenti del framework.

*   **`adapters.json`**: Schema per la configurazione degli adattatori.
*   **`transaction.json`**: Struttura di una transazione dati.
*   **`storekeeper.json`**: Configurazioni per il gestore della persistenza.
*   **`messenger.json`**: Configurazioni per il sistema di messaggistica.
*   **`tags.json`, `attributes.json`, `widgets.json`**: Probabilmente definizioni per il rendering UI o la validazione dei dati.
